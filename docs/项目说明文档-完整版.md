## 1. 项目介绍

`STEM_Agent` 是“洗衣机 STEM 项目制课程”的前端实验平台，包含科学探究与工程实践两条主线。

核心能力：
- 课程首页展示与导学视频播放
- 科学实验流程（10 步）
- 工程实验流程（9 步）
- AI 对话与语音引导
- 虚拟搭建/测试记录/报告生成
- 记录中心、成长中心、数据记录中心

当前已完成重构方向：
- 每个实验独立配置：`config/experiments/<id>.config.js`
- 实验素材统一目录：`static/experiments/<id>/images|videos`
- 实验目录与分类统一注册：`config/registry/*`

## 2. 技术与运行方式

技术栈：
- uni-app（Vue3）
- Vite 5
- H5 本地存储（草稿/进度/报告）

入口与路由：
- 程序入口：`main.js`、`App.vue`
- 路由配置：`pages.json`
- 页面目录：`pages/*`

## 11. 配置与页面关系（维护重点）

关键链路：
1. `config/experiments/*.config.js` 定义实验内容、语音、素材路径
2. `config/registry/experiment-registry.js` 注册实验
3. `config/registry/category-registry.js` 定义分类
4. `config/experiment-catalog.js` 生成首页目录
5. `pages/experiment/flow.vue` 与 `pages/engineering/flow.vue` 按实验 ID 加载模板

维护原则：
- 新增实验时先建独立配置文件，再注册到 registry。
- 只改配置不改组件，可最大化复用流程组件。
- 素材命名与路径保持稳定，避免页面层再次分叉。
